#西游超市
这是一个未上线的西游线上超市,包括 xiyou-mall 商城系统及 xiyou-mall-admin 商城后台管理系统，
基于前后端分离技术，后端基于 Spring Boot 及相关技术栈开发。 前台商城系统包含商品分类、商品信息
、商品搜索、购物车、订单、个人订单管理等模块。 后台管理系统包含用户数据面板、 商品管理、订单管理
、会员管理、分类管理、设置等模块。
